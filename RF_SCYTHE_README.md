# RF SCYTHE Homepage - Arapt.us

## Overview

This is the modernized Command Modern Ops-inspired homepage for the RF SCYTHE system, featuring ThreeJS-based visualizations and integration with all RF SCYTHE components.

## How to Access the Homepage

1. Start the server by running:
   ```bash
   ./start_rf_scythe_server.sh
   ```

2. Open your web browser and navigate to:
   ```
   http://localhost:8000/
   ```

## Features

- Command Modern Ops-inspired interface with ThreeJS visualizations
- Integrated status panel showing system component statuses
- Access to all RF SCYTHE modules from a centralized location
- Interactive 3D background with RF signal visualization
- Optimized performance with asynchronous loading
- JWST data integration with spectral analysis capabilities
- LHC RF visualization and hypergraph analysis

## System Requirements

- Modern web browser with WebGL support
- JavaScript enabled
- 4GB RAM minimum (8GB+ recommended for best performance)

## Troubleshooting

If the homepage doesn't appear:
1. Make sure the server is running (check terminal for errors)
2. Verify you're accessing http://localhost:8000/ and not https
3. Check your browser console for any JavaScript errors
4. Try clearing your browser cache
